print - команда которая выводит информацию в консоль
# - символ что бы указать комментарий в коде
+, -, /, *, %, <, >, <=, >= - математические операции
== - символ равенства(проверяет равны ли два значения)
%d, %s, %r - символы форматирования
end - оператор который добавляет в конец строчки что угодно
/a, /b, /f, /n, /N{имя}, /r, /t - символы управляющие последовательности
input() - функция которая принимает текст что написал пользователь в консоли
argv - переменная которая имеет список аргументов командной строчки
open() - функция открытия файла
read() - функция чтения файла
write() - функция записывания в файл
close() - функция закрытия файла
readline() - функция которая считывает только одну строку из файла
truncate() - Функция которая очищает файл
len() - функция которая получает длину строки
def {имя функции} (параметры) - пример написания функции
return - ключевое слово для возврата чего-нибудь из функции
